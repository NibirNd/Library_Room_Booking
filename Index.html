Overview

This project is a room reservation system that uses Google Forms, Sheets, and Apps Script to manage room bookings. Users submit a form to request a reservation, and the system checks for conflicts before sending a confirmation or conflict email to the user.

Step-by-step Guide

Step 1: Create the Google Form

  1. Create a new Google Form to collect reservation requests.
  2. Add fields for the user's name, email, ID, date, time, duration, and desired room.
  3. Make sure to set the appropriate field types (e.g., Date for the date field, Time for the time field, etc.).

Step 2: Link the Google Form to a Google Sheet

  1. In the Google Form, click on the "Responses" tab.
  2. Click on the Google Sheets icon to link the form to a new or existing Google Sheet.
  3. The form responses will now be saved in the linked Google Sheet.

Step 3: Create the Google Apps Script project

  1. Open the linked Google Sheet.
  2. Click on "Extensions" in the menu, then "Apps Script" to create a new Google Apps Script project.
  3. The script project will be linked to the Google Sheet.

Step 4: Write the Google Apps Script code

  1. Write the necessary Apps Script code to process the reservation requests, check for conflicts, and send emails to users. This includes functions such as Submission, getEndTime, getConflicts, draftEmail, and makeEmail.
  2. Use the onFormSubmit trigger to run the main function automatically when a new form response is submitted.
  3. Implement the necessary functions to format the date and time according to your desired timezone.

Step 5: Test the system

  1. Submit a few test reservations through the Google Form.
  2. Verify that the system is working correctly by checking the Google Sheet for reservation data and checking the emails sent to users.

Step 6: Share the project

  1. To share the project with others, click on the "Share" button in the upper-right corner of the Google Apps Script project.
  2. Enter the email addresses of the people you want to share the project with and set their permissions (e.g., "Can view" or "Can edit").
  3. Click on "Send" to share the project.

Conclusion

This room reservation system is a versatile and efficient way to manage room bookings using Google Forms, Sheets, and Apps Script. By following this guide, you can create a fully functional room reservation system tailored to your needs.

Feel free to modify the code, form fields, or other elements to suit the specific requirements of your organization or project.
